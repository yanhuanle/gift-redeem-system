## 一、功能实现情况

兑换码验证、礼包内容展示、历史记录三个页面UI部分开发完成，数据采用本地mock数据

## 二、技术选型及理由

选用了Vue + Vant框架进行开发。
对于前端框架，从React和Vue中二选其一，
对于Vant，是移动端比较常用的框架。
